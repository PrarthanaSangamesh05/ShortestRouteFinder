#  Karnataka Cities Route Finder

A Python GUI application that finds the **shortest route between cities in Karnataka** using **Dijkstra’s Algorithm**. The app provides a simple, user-friendly interface built with **Tkinter**.

---

##  Features
- Calculates shortest distance between two cities
- Displays full route and total distance
- Uses Dijkstra’s Algorithm with a priority queue
- Interactive GUI with error handling
- Bidirectional city graph

---

##  Tech Stack
- Python
- Tkinter (GUI)
- heapq (priority queue)
- Dijkstra’s Algorithm
